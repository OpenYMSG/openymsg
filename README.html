OpenYMSG README

OpenYMSG is an Java library that implements the Yahoo Instant Messenger protocol.

In this document, you'll find:

  • The history of OpenYMSG;
  • Building instructions;
  • A note on testing;
  • The boring stuff.

The history of OpenYMSG

OpenYMSG is based on jYMSG (http://jymsg9.sourceforge.net). The initial development goal was to improve that library. During discussions on how to achive this, different project goals surfaced. This resulted in a mutual agreement to branch from the jYMSG development tree, and start a new project. OpenYMSG was born.

To represent the branch and to acknowledge the original author(s) for his work, @author javadoc tags have been included in the orignal source files, and direct derivations thereof. As, at the time of the branch, S.E. Morris was the project lead of jYMSG, this name has been included in these tags. Where documented, other authors have been included as well, but for a complete list of original authors, please review the jYMSG home page.

Building instructions

The recommended way to build the OpenYMSG library is by using Apache Ant. An Ant build script has been included in the root directory of the project.

The default target of the Ant script, which you would run by executing ant from a command line interface in the root directory of this project, builds the complete library. Other targets are available. A list of them can be retrieved by issuing ant -projecthelp, again in the parent directory of this project.

A note on testing

Several JUnit testcases have been included in the project. Some of the included tests need to logon to the Yahoo network to verify the correctness of this library.

The Yahoo Network is known to be very unforgiving for automated logins. To circumvent potential problems, several different Yahoo accounts should be used to execute the testcases.

Yahoo testaccounts can be configured by modifying the yahooAuthenticationForJUnitTests.properties file in the resources directory. In it, several accounts can be configured that will be used to run the junit test. Please refer to that file for further information.
